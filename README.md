## 🐛 How to Report a Bug

Found something that isn't working right? I want to fix it! Please open a **New Issue** and select the **Bug Report** template. 

To help me squash the bug as fast as possible, please try to include:
1. **Device & OS:** (e.g., iPhone 15 Pro on iOS 17.4, or Pixel 7 on Android 14)
2. **App Version:** (e.g., v1.3.0+5)
3. **Steps to Reproduce:** Exactly what you clicked or typed before the bug happened.
4. **Expected Behavior vs. Actual Behavior:** What did you expect to happen, and what went wrong?
5. **Screenshots:** (If applicable)

## 💡 How to Request a Feature

Are you tired of a specific IT tool being locked behind a paywall in other apps? Do you have an idea that would make this app better? I'd love to hear it!

Please open a **New Issue** and select the **Feature Request** template. Let me know:
*   What the feature is.
*   How it would help your daily workflow.
